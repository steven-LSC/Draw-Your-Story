# Draw-Your-Story

Draw Your Story is an iOS APP, which you can draw or show your story, and save you story in the cloud.

### ScreenShot


### Features

This app have several features:

* Member management : Login, Register
* Forget password
* Cloud backup
* Automatically play hand-drawn pages


### Tech
* XCode 12
* SwiftUI
* FireBase
* CocoaPods
